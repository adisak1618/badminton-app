**Endpoints:**

**GET `/` — Fetch event + registration list (REG-02)**
- Query param: `eventId` (uuid, required via `t.Object({ eventId: t.String({ format: "uuid" }) })`)
- Look up current member from `session.lineUserId` -> `members` table
- Fetch event by eventId; if not found throw `notFound("Event")`
- Fetch all registrations for this eventId JOINed with members to get `displayName`
- Look up caller's role in event's club via `requireClubRole` wrapped in try/catch — if throws forbidden, set `isAdmin = false`; if returns "owner" or "admin", set `isAdmin = true`
- Find current member's registration in list to get `currentMemberRegistrationId`
- Return:
```json
{
  "event": { "id", "title", "eventDate", "venueName", "venueMapsUrl", "status", "maxPlayers", "clubId" },
  "registrations": [{ "id", "memberId", "displayName" }],
  "registeredCount": <number>,
  "currentMemberRegistrationId": <string|null>,
  "isAdmin": <boolean>
}
```

**POST `/` — Register member (REG-01, BOT-02)**
- Body: `t.Object({ eventId: t.String({ format: "uuid" }) })`
- Look up member from `session.lineUserId`
- Fetch event; validate `event.status === "open"` — if not, return `new ApiError(409, "EVENT_CLOSED", "ปิดรับลงทะเบียนแล้ว")`
- Count current registrations: `SELECT COUNT(*) FROM registrations WHERE event_id = eventId`
- If `count >= event.maxPlayers`, return `new ApiError(409, "EVENT_FULL", "งานนี้เต็มแล้ว")`
- INSERT into registrations `{ eventId, memberId: member.id }`
- Catch unique constraint error (Postgres code `23505`): return `new ApiError(409, "ALREADY_REGISTERED", "ลงทะเบียนแล้ว")`
- After successful INSERT, re-count registrations to get accurate `newCount`
- Call `repostFlexCard` helper (defined below) with action "register", member.displayName, newCount
- Return 201 with `{ id: registration.id, registeredCount: newCount }`

**DELETE `/:registrationId` — Cancel own or admin remove (REG-03, REG-05)**
- Param: `registrationId` (uuid)
- Fetch registration by id; if not found, throw `notFound("Registration")`
- Fetch event by `registration.eventId`
- Look up caller member from `session.lineUserId`
- If `registration.memberId !== callerMember.id`:
  - Check admin: `await requireClubRole(event.clubId, callerMember.id, ["owner", "admin"])` — if throws, re-throw (403)
  - This is admin remove; set `isAdminRemove = true`
- DELETE the registration row
- Re-count registrations for accurate `newCount`
- Fetch removed member's displayName (for non-admin-remove repost alt text)
- If `isAdminRemove`: call `repostFlexCard` with action "admin_remove"
- Else: call `repostFlexCard` with action "cancel" and removed member's displayName
- Return 200 with `{ registeredCount: newCount }`

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | POST /registrations | mitigate | Validate event.status === "open" AND registeredCount < maxPlayers server-side before INSERT |
| T-05-02 | Tampering | DELETE /registrations/:id | mitigate | Verify registration.memberId === session.memberId OR caller has admin role via requireClubRole |
| T-05-03 | Elevation of Privilege | PATCH /events/:id/status | mitigate | requireClubRole(event.clubId, memberId, ["owner", "admin"]) before status mutation |
| T-05-04 | Tampering | POST /registrations duplicate | mitigate | DB unique constraint (eventId, memberId) catches; map PG error 23505 to 409 |
| T-05-05 | Denial of Service | POST /registrations race condition | accept | Read-then-write count check acceptable at casual badminton group scale; DB unique prevents duplicates |
| T-05-06 | Information Disclosure | GET /registrations | accept | Registration list (displayNames) is visible to all authenticated club members — intentional per REG-02 |
| T-05-07 | Spoofing | All endpoints | mitigate | authMiddleware validates iron-session cookie; session.lineUserId used for member lookup, never client-supplied |
