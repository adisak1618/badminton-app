Create `apps/web/app/liff/events/[id]/register/page.tsx` — a "use client" Next.js page following the exact layout pattern of `events/create/page.tsx`.

**Page structure:**

```typescript
"use client";

import { Suspense, useCallback, useEffect, useState } from "react";

import { toast } from "sonner";
```

**Default export** wraps inner component in `<Suspense>` with Loader2 fallback (same pattern as create page).

**Inner component `LiffEventRegisterPage`:**

**State:**
- `loading: boolean` — true until first fetch completes
- `submitting: boolean` — true during register/cancel/remove/close actions
- `error: string | null` — error message
- `event: EventData | null`
- `registrations: Registration[]`
- `registeredCount: number`
- `currentMemberRegistrationId: string | null`
- `isAdmin: boolean`

**Types (define inline in file):**
```typescript
interface EventData {
  id: string; title: string; eventDate: string; venueName: string | null;
  venueMapsUrl: string | null; status: "open" | "closed" | "cancelled"; maxPlayers: number; clubId: string;
}
interface Registration {
  id: string; memberId: string; displayName: string;
}
```

**Data fetching:**
- `const { id: eventId } = useParams<{ id: string }>();`
- `fetchData` function: GET `/api/proxy/registrations?eventId=${eventId}`, parse response, set all state
- `useEffect` with `[isReady, isLoggedIn]` deps: guard `if (!isReady || !isLoggedIn) return;`, call `fetchData()`, also add `visibilitychange` listener per D-04:
```typescript
const handleVisibility = () => {
  if (document.visibilityState === "visible") fetchData();
};
document.addEventListener("visibilitychange", handleVisibility);
return () => document.removeEventListener("visibilitychange", handleVisibility);
```

**Computed state:**
- `const isFull = registeredCount >= (event?.maxPlayers ?? 0);`
- `const isClosed = event?.status === "closed";`
- `const isRegistered = currentMemberRegistrationId !== null;`

**Action handlers:**

`handleRegisterToggle`:
- If `isRegistered`: DELETE `/api/proxy/registrations/${currentMemberRegistrationId}`
  - On success: toast.success("ยกเลิกสำเร็จ"), call `fetchData()`
  - On 409: toast.error(response error message)
- Else: POST `/api/proxy/registrations` with `{ eventId }`
  - On 201: toast.success("ลงทะเบียนสำเร็จ"), call `fetchData()`
  - On 409: toast.error based on response code — "ลงทะเบียนแล้ว" for ALREADY_REGISTERED, "งานนี้เต็มแล้ว" for EVENT_FULL, "ปิดรับลงทะเบียนแล้ว" for EVENT_CLOSED
  - On other error: toast.error("ลงทะเบียนไม่สำเร็จ กรุณาลองใหม่")
- Set `submitting = true` before, `false` after (in finally)

`handleRemoveMember(registrationId: string)`:
- DELETE `/api/proxy/registrations/${registrationId}`
- On success: toast.success("ลบสำเร็จ"), call `fetchData()`
- On error: toast.error("ลบไม่สำเร็จ")

`handleToggleClose`:
- PATCH `/api/proxy/events/${eventId}/status` with `{ status: isClosed ? "open" : "closed" }`
- On success: toast.success(isClosed ? "เปิดรับลงทะเบียนสำเร็จ" : "ปิดรับลงทะเบียนสำเร็จ"), call `fetchData()`

**Render (follows UI-SPEC layout exactly):**

Loading state: full-screen centered `<Loader2 className="h-8 w-8 animate-spin text-muted-foreground" aria-label="Loading" />`

Error state: `<div className="max-w-sm mx-auto px-4 py-8"><p className="text-destructive">{error}</p></div>`

Main layout:
```jsx

  <h1 className="text-2xl font-bold mb-6">ลงทะเบียน</h1>

  {/* Full state badge — per D-14 */}
  {isFull && !isClosed && (
    <span className="inline-block mb-4 bg-destructive text-destructive-foreground px-2 py-1 rounded text-sm">
      เต็มแล้ว
    </span>
  )}

  {/* Event info card */}
  <Card className="mb-6">
    <CardContent className="pt-6 space-y-1">
      <p className="text-sm font-bold">{event.title}</p>

      {event.venueName && <p className="text-sm">{event.venueName}</p>}
      <p className={`text-sm ${isFull ? "text-destructive" : "text-foreground"}`}>
        {registeredCount}/{event.maxPlayers} คน
      </p>
    </CardContent>
  </Card>

  {/* Register/Cancel button — per D-01, D-03, D-14, D-16 */}
  <Button
    className="w-full min-h-[44px] mb-6"
    disabled={submitting || (isFull && !isRegistered) || isClosed}
    onClick={handleRegisterToggle}
  >
    {submitting ? (
      <>
        <Loader2 className="mr-2 h-4 w-4 animate-spin" />
        {isRegistered ? "กำลังยกเลิก..." : "กำลังลงทะเบียน..."}
      </>
    ) : isClosed ? (
      "ปิดรับลงทะเบียนแล้ว"
    ) : isFull && !isRegistered ? (
      "เต็มแล้ว"
    ) : isRegistered ? (
      "ยกเลิก"
    ) : (
      "ลงทะเบียน"
    )}
  </Button>

  {/* Member list — per D-02 */}
  <Card>
    <CardContent className="pt-6">
      <p className="text-sm font-bold mb-4">ลงทะเบียนแล้ว</p>
      {registrations.length === 0 ? (

                <button
                  className="text-muted-foreground hover:text-destructive p-1"
                  onClick={() => handleRemoveMember(reg.id)}
                  aria-label={`ลบ ${reg.displayName}`}
                >
                  <X className="h-4 w-4" />
                </button>
              )}
            </li>
          ))}
        </ul>

  {isAdmin && (
    <Button

      disabled={submitting}
      onClick={handleToggleClose}
    >
      {isClosed ? "เปิดรับลงทะเบียน" : "ปิดรับลงทะเบียน"}
    </Button>
  )}
</div>
```
