| Token | Value | Usage |

## Page Layout Contract

### Club Hub (`/clubs/[id]` — server component)

Top-to-bottom section order:

1. **Club header** — club name (Display, 28px/600) + role badge + LINE link status badge + Settings/Members quick-link buttons (owner/admin only, `variant="outline"`)
2. **Separator**
3. **Weekly Schedule section** — Card with title "ตารางซ้อม" + list of active event templates shown as rows (day + time + venue). Displayed ABOVE upcoming events per D-05.
4. **Upcoming Events section** — heading "อีเวนต์ที่กำลังจะมาถึง" + vertical list of event cards, sorted chronologically ascending.
5. No sidebar. Single-column on mobile. Two-column event card grid only if viewport >= md breakpoint (768px).

### Event Card (within events list)

Each card contains in order:
- **Title** (Heading, 20px/600) — event date + day formatted as "วันเสาร์ที่ 18 เม.ย. 2026 19:00"
- **Venue** (Body, 16px/400) — venue name
- **Player count progress bar** — visual fill bar + fraction text "X/Y คน" (Label, 14px/400) beside bar
- **Fees row** (Label, 14px/400, muted-foreground) — "ลูก X฿ · สนาม X฿"
- **Register button** — full-width, `variant="default"` (accent/primary), min-height 44px, text "ลงทะเบียน"

### Weekly Schedule Row (within schedule section)

Each template rendered as a single flex row:
- Thai day name (Label, 14px/600 semibold) — "จันทร์", "อังคาร", etc.
- Start time (Label, 14px/400) — "19:00"
- Venue name (Label, 14px/400, muted-foreground) — truncated with ellipsis if long

### LIFF Club Hub (`/(liff)/clubs/[id]` — client component)

Same visual layout as website version. No `<Nav />` bar — uses `(liff)` route group layout with LiffProvider. Register button links to existing `/(liff)/events/[id]` for registration flow.

## Interaction States

### Events List

| State | UI |
|-------|----|
| Loading | Full-screen centered `Loader2` spinner (h-8 w-8, muted-foreground) |
| Loaded — has events | Vertical card list, chronological order |
| Loaded — no events | Empty state (see Copywriting Contract) |
| API error | Error state (see Copywriting Contract) |

### Event Card — Register Button

| State | UI |
|-------|----|
| Registration open | "ลงทะเบียน" button, `variant="default"`, enabled |
| Event full (registered = max) | Button replaced with Badge `variant="secondary"` text "เต็มแล้ว", no register action |
| Registration closed | Button replaced with Badge `variant="outline"` text "ปิดรับสมัคร" |

### Progress Bar

- Filled portion width = `(registered / maxPlayers) * 100%`, clamped 0–100%
- Fill color: `--primary` (near-black)
- Track color: `--muted`
- Height: 8px, border-radius: 4px (pill)
- Fraction text "X/Y คน" displayed inline to the right of the bar at Label size

### Weekly Schedule Section

| State | UI |
|-------|----|
| Has active templates | Rows listed |
| No active templates | Section hidden entirely — do not show empty schedule card |
