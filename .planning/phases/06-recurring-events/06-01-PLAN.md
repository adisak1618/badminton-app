3. Create `apps/api/src/routes/event-templates.ts` with these routes:

- `GET /event-templates?clubId=:id` — list all active templates for a club. Requires authMiddleware + requireClubRole(clubId, memberId, ["owner","admin"]).

- `POST /event-templates` — create template. Body: `{ clubId, venueName, venueMapsUrl?, shuttlecockFee, courtFee, maxPlayers, title?, eventDayOfWeek, eventTime, openDayOfWeek, openTime }`. Validate with Elysia `t.Object`. Insert into `eventTemplates` table. Return 201 with template object.

- `PATCH /event-templates/:id` — update template. Per D-08, changes only affect future occurrences (only updates the template row). Per D-09, if body contains `maxPlayers`, check all events with `templateId = :id` AND `status IN ('draft','open')` — count registrations for each. If any event has `registrationCount > newMaxPlayers`, return 422 with `{ error: "จำนวนผู้เล่นสูงสุดน้อยกว่าผู้ที่ลงทะเบียนอยู่แล้ว" }`.

- `POST /event-templates/:id/create-now` — per D-04 "create now" mode. Reads the template, generates a one-time event record from its fields (copies venueName, fees, maxPlayers, etc.). Sets `eventDate` to the next occurrence of `eventDayOfWeek` at `eventTime` in Asia/Bangkok. Sets `templateId` on the event. Sets event status to `open`. Looks up club's `lineGroupId`, builds Flex Message via `buildEventFlexCard`, pushes to group via `lineClient.pushMessage`. Returns 201 with the created event.

- `PATCH /event-templates/:id/occurrences/:eventId/cancel` — per D-12. Sets event status to `cancelled`. Builds `buildCancellationFlexCard` with event data. Pushes cancellation Flex Message to club's LINE group. Returns 200 with updated event.

All routes use `authMiddleware` and `requireClubRole` with `["owner", "admin"]`.

Use `date-fns-tz` for timezone arithmetic: `toZonedTime(new Date(), "Asia/Bangkok")` to get current Bangkok time, `fromZonedTime(bangkokDate, "Asia/Bangkok")` to convert back to UTC for `eventDate`.
