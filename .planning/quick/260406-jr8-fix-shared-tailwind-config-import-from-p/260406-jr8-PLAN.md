Current content:
```css

```

Replace with:

Note on `@source`: Tailwind v4's `@source` directive does NOT support node module resolution
(it only understands filesystem glob paths). Keep the relative path for `@source` -- this is
the correct approach per Tailwind v4 docs. Only the `@import` benefits from package path
resolution via PostCSS.

The `@repo/ui` package.json already exports `"./styles/*": "./src/styles/*.css"`, so the
PostCSS import resolver will find `packages/ui/src/styles/theme.css` through the pnpm
workspace symlink at `apps/web/node_modules/@repo/ui`.
